# Cat Cafe Idle Game

A delightful idle clicker game built with Flutter where you run your own cat-themed coffee shop! Brew "Catpuccinos", train adorable server cats, and recruit cafe cats to automate your coffee empire.

## Features

- **Tap to Brew**: Click the brew button to make Catpuccinos
- **Server Cat Training**: Upgrade your brewing power to earn more per click
- **Cafe Cat Recruitment**: Hire cats that automatically brew Catpuccinos every second
- **Random Cat Visitors**: Special cats visit every 25 seconds bringing gifts (with a chance for rare bonuses)
- **Progressive Upgrades**: Costs increase exponentially to keep the game challenging
- **Clean UI**: Simple, intuitive interface with a cute cat cafe theme

## How to Play

1. **Brew Catpuccinos**: Tap the "Brew Catpuccino" button to earn your first cups
2. **Train Server Cats**: Spend 20 Catpuccinos to increase your brewing power by +1 per tap
3. **Recruit Cafe Cats**: Spend 50 Catpuccinos to hire cats that brew +1 Catpuccino per second
4. **Keep Upgrading**: The more you upgrade, the faster you earn!
5. **Reset When Ready**: Close the cafe for cleaning to start fresh

## Game Mechanics

- **Starting Resources**: Begin with 0 Catpuccinos
- **Base Click Power**: 1 Catpuccino per tap
- **Server Cat Training**:
  - Initial cost: 20 Catpuccinos
  - Cost multiplier: 1.5x per level
  - Adds +1 Catpuccino per click
- **Cafe Cat Recruitment**:
  - Initial cost: 50 Catpuccinos
  - Cost multiplier: 1.6x per level
  - Adds +1 Catpuccino per second
- **Random Cat Visitors**:
  - Arrive every 25 seconds
  - Normal gift: 10-30 Catpuccinos
  - Rare gift (10% chance): 100 Catpuccinos
